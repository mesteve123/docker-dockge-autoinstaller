# Docker & Dockge Auto Installer

Install Docker and Dockge with a single script.

## Features
- Works on **Ubuntu 20.04 / 22.04 / 24.04**
- Dockge starts automatically on system boot
- Access to Dockge **HTTP (port 5001)**

## Installation
Just run:
```bash
curl -fsSL https://raw.githubusercontent.com/mesteve123/docker-dockge-autoinstaller/main/install_docker_dockge.sh -o install_docker_dockge.sh && chmod +x install_docker_dockge.sh && sudo ./install_docker_dockge.sh
